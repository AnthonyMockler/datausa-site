API

Documentation on how to build your own apps using data powered by the DataUSA API.

Quick Start

Introduction

At its core, the DataUSA API allows users to show data across various levels of detail, called sumlevels. There are four core categories of data: geographies, occupations, industries and educational studies. We will begin with a quick look at exploring geographic data.

Exploring Population Data

First steps

To get population data at the national level we can use the following API call:

http://db.datausa.io/api?show=geo&sumlevel=nation&required=pop

This API call however, will return data for all available years in the database. To only retrieve the latest year’s data, use:

http://db.datausa.io/api?show=geo&sumlevel=nation&year=latest&required=pop

This call gives us the popoulation of the US based on the latest available year of data. To get data at the state level we can use:

http://db.datausa.io/api?show=geo&sumlevel=state&year=latest&required=pop
